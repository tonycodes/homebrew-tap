# Homebrew Tap for TonyCodes

## Installation

```bash
brew tap tonycodes/tap
```

## Available Formulae

### CopilotGuard

A lightweight daemon that intercepts and analyzes AI coding assistant traffic.

```bash
brew install tonycodes/tap/copilotguard
```

After installation, complete setup with:

```bash
sudo copilotguard install
```
